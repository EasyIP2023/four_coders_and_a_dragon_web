Four Coders and a Dragon

Purpose

To design and develop a modular robot that fulfills multiple functionalities based upon the userâ€™s needs. The technology to be developed must be able to adapt to changing functional needs. Modularity is paramount, and therefore is to be emphasized throughout the entirety of the project lifecycle. Once software development is completed, the user should only be responsible for adding or removing parts from the robot, allowing the robot to gain or modify its functionality to meet a new need. The goal for this project is to develop the robot to fit a single use case and developing differing functions to accomplish this use case to meet modularity requirements.
